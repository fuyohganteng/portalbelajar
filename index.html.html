Pengenalan Komputer

Dasar-dasar hardware & software.

Mulai Belajar
Apa itu Komputer?

Komputer adalah perangkat elektronik yang dapat memproses data, menerima input, menyimpan data, dan menghasilkan output. Ini adalah alat yang sangat penting di era digital.

Bagian-bagian Utama Komputer:
  • **Hardware:** Bagian fisik komputer seperti CPU, RAM, Hard Drive, Monitor, Keyboard, Mouse.
  • **Software:** Program atau aplikasi yang berjalan di komputer, contohnya sistem operasi (Windows, MacOS), browser (Chrome), atau aplikasi pengolah kata.

Dasar Pemrograman

Belajar logika coding sederhana (Python/Scratch).

Mulai Belajar
Memulai Dunia Coding

Pemrograman adalah proses menulis instruksi yang dapat diikuti oleh komputer untuk melakukan tugas tertentu. Ini adalah bahasa yang kita gunakan untuk berbicara dengan komputer.

Konsep Dasar:
  • **Algoritma:** Langkah-langkah terstruktur untuk menyelesaikan masalah.
  • **Variabel:** Tempat menyimpan data.
  • **Kondisi (If/Else):** Membuat keputusan dalam kode.
  • **Loop (Perulangan):** Mengulang tugas secara otomatis.

Jaringan & Internet

Memahami cara kerja dunia maya.

Mulai Belajar
Bagaimana Internet Bekerja?

Internet adalah jaringan komputer global yang memungkinkan miliaran perangkat saling terhubung dan berbagi informasi di seluruh dunia.

Konsep Kunci:
  • **Router:** Perangkat yang mengarahkan lalu lintas data.
  • **IP Address:** Alamat unik setiap perangkat di jaringan.
  • **Web Server:** Komputer yang menyimpan website dan mengirimkannya ke browser kita.

Keamanan Siber

Melindungi diri di dunia digital.

Mulai Belajar
Lindungi Dirimu di Dunia Maya!

Keamanan siber adalah praktik melindungi sistem komputer, jaringan, dan data dari pencurian, kerusakan, atau akses tidak sah.

Tips Penting:
  • **Kata Sandi Kuat:** Gunakan kombinasi huruf, angka, simbol, dan jangan mudah ditebak.
  • **Waspada Phishing:** Jangan klik tautan atau unduh lampiran dari email mencurigakan.
  • **Antivirus:** Pastikan perangkatmu memiliki software antivirus yang aktif.

Pengantar Robotika

Mengenal dasar robot dan AI.

Mulai Belajar
Robot: Teman atau Musuh?

Robot adalah mesin yang diprogram untuk melakukan tugas secara otomatis. Robotika adalah bidang yang mempelajari desain, konstruksi, operasi, dan aplikasi robot.

Elemen Robot:
  • **Sensor:** Untuk merasakan lingkungan (mata, telinga elektronik).
  • **Aktuator:** Untuk melakukan gerakan (motor, roda).
  • **Kontroler:** Otak robot (mikrokomputer).

Sentuhan Visual di Era Digital

Desain grafis adalah seni komunikasi visual yang menggunakan tipografi, fotografi, ilustrasi, dan desain layout untuk menyampaikan pesan atau ide.

Prinsip Dasar Desain:

Proyek 1: Membuat Halaman Web Sederhana (HTML & CSS)

Deadline: 5 Hari

Mulai Proyek
Deskripsi Proyek:

Buatlah sebuah halaman web statis sederhana menggunakan HTML untuk struktur dan CSS untuk styling. Tema bebas (misalnya: profil pribadi, hobi, resep).

Langkah-langkah Proyek:
  1. Buat file `index.html` dan `style.css`.
  2. Definisikan struktur dasar HTML (header, main, footer).
  3. Tambahkan konten berupa teks dan gambar.
  4. Terapkan styling dasar menggunakan CSS (warna, font, layout sederhana).
  5. Pastikan halaman responsif untuk perangkat mobile.

Kumpulkan hasilnya melalui [Link Drive berikut].

Latihan Debugging Code Python

Deadline: 3 Hari

Mulai Latihan
Deskripsi Latihan:

Diberikan potongan kode Python yang mengandung beberapa kesalahan (bug). Tugas Anda adalah menemukan dan memperbaiki semua bug tersebut hingga kode berjalan dengan benar.

Kode Python yang Disediakan:

def hitung_rata_rata(angka):
    total = 0
    for i in angka:
        total += i
    return total / len(angka)

data = [10, 20, 30, 40, 50]
print("Rata-rata:", hitung_rata_rata(data))

Temukan bug-nya dan jelaskan perbaikannya. Kirim kode yang sudah diperbaiki ke [Link Drive berikut].

Studi Kasus: Analisis Keamanan Akun Online

Deadline: 7 Hari

Mulai Studi Kasus
Deskripsi Studi Kasus:

Pilih satu platform online yang sering Anda gunakan (misalnya: media sosial, e-commerce, email). Analisis fitur keamanan yang disediakan dan identifikasi potensi kerentanan keamanan yang mungkin ada. Berikan saran perbaikan.

Poin Analisis:
  • Autentikasi dua faktor (2FA)
  • Kebijakan kata sandi
  • Pemberitahuan login mencurigakan
  • Enkripsi data
  • Penanganan data pribadi

Buat laporan singkat (maksimal 2 halaman) dan kumpulkan.

Proyek 2: Desain Jaringan Sederhana

Deadline: 6 Hari

Mulai Proyek
Deskripsi Proyek:

Desain topologi jaringan sederhana untuk sebuah kantor kecil yang memiliki 1 server, 5 komputer karyawan, 1 printer jaringan, dan akses internet. Gambarkan diagramnya dan jelaskan komponen yang dibutuhkan.

Aspek yang Perlu Diperhatikan:
  • Jenis koneksi internet (contoh: Fiber Optic, ADSL)
  • Perangkat jaringan (router, switch, kabel)
  • Penempatan perangkat
  • Saran keamanan dasar jaringan

Serahkan diagram dan penjelasan Anda.
